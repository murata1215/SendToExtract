SendToExtract

右クリック → 送る → 一括展開

Windows の「送る」メニューからアーカイブをまとめて展開するツール

無料ダウンロード

Windows 10 / 11 (64bit) | v1.0.2 | 約65MB | .NET不要

特徴

📦

多形式対応

zip, tar.gz, 7z, rar など主要なアーカイブ形式をすべてサポート

🖱

かんたん操作

右クリック→「送る」で一括展開。ファイル選択画面で確認してから展開

🇯🇵

日本語対応

CP932（Shift_JIS）のファイル名も文字化けなし

⚡

インストール不要

exe 1つで動作。.NET ランタイムは不要。好きな場所に置くだけ

使い方

ダウンロード & セットアップ

zip を解凍して SendToExtract.exe を好きな場所に配置。
ダブルクリックして「送る」に登録ボタンを押すだけ。

右クリック →「送る」

展開したいフォルダやファイルを右クリックし、
「送る」メニューから「SendToExtract」を選択。

ファイルを選択

フォルダ内のアーカイブが一覧表示されます。
展開するファイルをチェックして「展開」ボタン。

展開完了

進捗バーで展開状況を確認。
完了後は「フォルダを開く」で展開先を確認できます。

TAR+BZip2 .tar.bz2 .tbz2
TAR+XZ .tar.xz .txz
7-Zip .7z
RAR .rar
GZip .gz
BZip2 .bz2

設定

%APPDATA%\SendToExtract\settings.json で設定を変更できます

設定項目初期値説明
collisionPolicyRename展開先が既にある場合の処理（Skip / Rename / Overwrite）
flattenSingleRoottrue二重ラップ平坦化（name/name/... → name/...）
recurseSubfoldersfalseサブフォルダも再帰的に探索
